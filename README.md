# 📱 AppBibleNative

Um pequeno aplicativo desenvolvido com **React Native utilizando Expo** com o objetivo de demonstrar o uso de componentes básicos da interface.

Este projeto possui **uma única tela inicial** contendo diferentes componentes comuns no desenvolvimento mobile. Todos os elementos estão organizados dentro de um **ScrollView**, permitindo a rolagem vertical da tela.

O objetivo principal deste aplicativo é **praticar e compreender o funcionamento de componentes essenciais do React Native**, além de entender como estruturar uma interface simples.

---

# 🎯 Objetivo da atividade

Criar um aplicativo mobile simples que apresente alguns dos principais componentes de interface do React Native.

A tela inicial reúne diferentes elementos visuais e interativos para demonstrar como cada componente funciona dentro de uma aplicação real.

Todos os componentes estão organizados dentro de um **ScrollView**, permitindo que o usuário navegue pela tela através da rolagem vertical.

---

# 🧩 Componentes utilizados

O aplicativo contém os seguintes componentes:

### ActivityIndicator

Exibe um indicador visual de carregamento, geralmente utilizado para informar ao usuário que algum processo está em andamento.

### Button

Botão interativo que executa uma ação quando pressionado.

### Text

Componente responsável por exibir textos na tela.

### TextInput

Campo de entrada onde o usuário pode digitar informações.

### Image (imagem local)

Imagem armazenada dentro do próprio projeto e exibida na interface.

### Image (imagem Local)

Imagem armazenada dentro do próprio projeto e exibida na interface.

### ScrollView

Componente responsável por permitir **rolagem vertical da tela**, garantindo que todos os elementos possam ser visualizados mesmo em telas menores.

# 🚀 Como executar o projeto

1. Instalar as dependências:

```
npm install
```

2. Iniciar o projeto com Expo:

```
npx expo start
```

3. Após iniciar, você pode abrir o aplicativo:

* no **emulador Android**
* no **simulador iOS**
* ou no aplicativo **Expo Go** no celular

# 💡 Considerações

Este projeto foi desenvolvido com fins educacionais para praticar conceitos básicos de desenvolvimento mobile com React Native.

Apesar de simples, ele demonstra a estrutura fundamental de um aplicativo, além do uso de componentes essenciais para construção de interfaces.

Com base neste projeto é possível evoluir para aplicações mais completas, adicionando navegação entre telas, integração com APIs e novos componentes de interface.
